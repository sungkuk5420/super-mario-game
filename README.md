# super-mario-game
Kaboom.js를 사용한 슈퍼마리오 게임입니다.

Kaboom.jsを使ったスーパーマリオゲームです。
## Demo / 데모 / デモサイト
[Link Open](https://super-mario-game.netlify.app/)
  

## Getting Started / 어떻게 시작하나요? / 始め方
### Installing / 설치 / インストール

```
npm install 
```
### Run / 실행 / 実行
 
```
node app.js
```
